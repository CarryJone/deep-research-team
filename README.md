# Skill Collection

This repository contains two standalone skills:

- [`deep-research-team/`](./deep-research-team) for multi-agent deep research workflows
- [`prompt-master/`](./prompt-master) for generating sharper prompts for AI tools

## Repository Layout

```text
.
├── README.md
├── deep-research-team/
│   ├── README.md
│   ├── SKILL.md
│   └── prompts/
└── prompt-master/
    ├── README.md
    ├── SKILL.md

```

Each folder is self-contained and can be used as its own skill.
